# 📱 Modern Login Screen

A modern Android Login UI built using **Java** in **Android Studio**. This project demonstrates a clean and user-friendly login interface with custom UI components and activity navigation.

## ✨ Features

- Modern Login Screen UI
- Custom Buttons
- Custom EditText Design
- Card Layout
- Image Support
- Activity Navigation
- Clean Material-inspired Design

## 🛠️ Tech Stack

- Java
- Android Studio
- XML
- Android SDK

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/sim-rah/Modern-Login-Screen.git
```

### Open Project

Open the project in **Android Studio**.

### Run

- Sync Gradle
- Connect an Android device or Emulator
- Click **Run ▶️**
